# Overview

## 프로그램 개요

PPOP Blog는 네이버 블로그 자동화 프로그램입니다. 키워드를 자동으로 추출하고, 글과 이미지로 구성된 포스팅을 자동으로 발행하는 기능을 제공합니다.

## 주요 기능

### 키워드 자동 추출
- 블로그 포스팅에 적합한 키워드를 자동으로 추출합니다
- 키워드 기반으로 포스팅 주제를 선정할 수 있습니다

### 자동 포스팅 생성
- 추출된 키워드를 기반으로 글을 자동 생성합니다
- 이미지를 첨부하여 완성된 포스팅을 구성합니다

### 자동 발행
- 네이버 블로그에 로그인하여 포스팅을 자동으로 업로드합니다
- 사용자 검토 후 발행 여부를 선택할 수 있습니다

## 프로그램 목적

블로그 운영의 반복적인 작업을 자동화하여 효율성을 높이고, 키워드 기반의 체계적인 콘텐츠 관리를 지원합니다.
