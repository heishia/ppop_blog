# Requirements

## 기능 모듈

프로그램은 총 5개의 주요 기능 모듈로 구성됩니다.

### 1. 키워드 추출 모듈 (keyword)

**기능**
1. **검색어로 일반 키워드 데이터 조회**
   - 검색어를 입력하여 일반 키워드의 데이터를 조회합니다
   - 조회수, 문서량 등 키워드 관련 데이터를 제공합니다

2. **주제 기반 황금키워드 추출**
   - 주제(subject)를 선택하여 황금키워드를 추출합니다
   - 주제 선택 시 연관키워드를 조회하고 필터를 거쳐 황금키워드를 자동으로 추출합니다

**키워드 종류**
- **황금키워드 (golden)**: 검색량과 문서수 조건을 필터링했을 때 나온 키워드
- **일반키워드 (normal)**: 일반적인 키워드
- **연관키워드 (related)**: 주제 선택 시 조회되는 연관 키워드 (프로그램 로직상 연관키워드를 조회한 후 필터를 거쳐 황금키워드를 추출하기 위해 구분)

**요구사항**

**검색어로 일반 키워드 데이터 조회**
- 검색어 입력을 통한 일반 키워드 검색
- 키워드 조회수 조회
- 키워드 문서량 조회
- 키워드 검색량 조회
- 키워드 관련 메타데이터 조회

**주제 기반 황금키워드 추출**
- 주제(subject) 선택 기능
- 주제 선택 시 연관키워드 자동 조회
- 연관키워드 필터링 (검색량 및 문서수 조건 검증)
- 황금키워드 자동 추출 및 분류
- 황금키워드 목록 조회
- 황금키워드 조건 설정 및 관리

### 2. 글 작성 모듈 (script)

**기능**
- 추출된 키워드를 기반으로 블로그 포스팅용 글을 작성합니다
- 작성된 글을 관리하고 검토할 수 있습니다

**요구사항**
- 글 목록 조회 (페이지네이션 지원)
- 특정 글 조회
- 새 글 생성
- 글 내용 수정
- 글 삭제

### 3. 이미지 첨부 모듈 (image)

**기능**
- 포스팅에 첨부할 이미지를 업로드하고 관리합니다
- 이미지 파일을 저장하고 메타데이터를 관리합니다

**요구사항**
- 이미지 파일 업로드
- 이미지 정보 생성
- 이미지 목록 조회 (페이지네이션 지원)
- 특정 이미지 조회
- 이미지 정보 수정
- 이미지 삭제

### 4. 로그인 모듈 (login)

**기능**
- 네이버 블로그에 로그인하는 기능을 제공합니다
- 로그인 세션을 관리합니다

**요구사항**
- 네이버 블로그 로그인 처리
- 로그인 상태 확인
- 세션 관리

### 5. 포스팅 업로드 모듈 (upload)

**기능**
- 완성된 포스팅(글 + 이미지)을 네이버 블로그에 업로드합니다
- 업로드 이력과 상태를 관리합니다

**요구사항**
- 포스팅 업로드 처리
- 업로드 목록 조회 (페이지네이션 지원)
- 특정 업로드 이력 조회
- 업로드 삭제

## 비기능 요구사항

- FastAPI 기반 RESTful API 제공
- Supabase 데이터베이스 사용
- 로깅 및 에러 처리
- 타입 힌트를 활용한 코드 작성
- 모듈별 독립적인 구조 (router/service/schemas)
